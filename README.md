# 🎯 Student Score Predictor

A simple and effective machine learning project that predicts a student's final score using a Linear Regression model. This project is built around a student performance dataset and demonstrates how data science can be used to estimate academic outcomes from basic features.

## ✨ Project Overview

This model analyzes important academic factors such as:
- Study hours
- Attendance
- Number of assignments completed

Using these inputs, the model predicts the likely final score of a student.

## 🧠 What the Model Does

The project uses a regression-based approach to learn patterns from historical student data and make predictions for new students. It is a great beginner-friendly example of:
- Data loading and preprocessing
- Feature selection
- Model training
- Prediction
- Performance evaluation

## 📊 Dataset

The project uses the dataset file [student_performance.csv](student_performance.csv), which contains student-related features and corresponding score values.

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## 📁 Project Files

- [student_performance.csv](student_performance.csv) - Student performance dataset
- [ml.ipynb](ml.ipynb) - Notebook for training and prediction
- [predict student score model.ipynb](predict%20student%20score%20model.ipynb) - Additional notebook for the model workflow

## ▶️ How to Use

1. Open the notebook in Jupyter.
2. Run the cells step by step.
3. Provide student feature values such as hours, attendance, and assignments.
4. View the predicted score generated by the model.
